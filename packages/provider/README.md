# @overledger/provider

Class used for the [Overledger SDK][repo] Gateway connection.

## Installation

Install using [npm](https://www.npmjs.org/):
```
npm install @overledger/provider
```

Or, if you prefer using [yarn](https://yarnpkg.com/):

```
yarn add @overledger/provider
```
